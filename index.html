    Version 2

    This is version 2 of mdv, available on the CLI as mdv2. v2 is a complete rewrite, offering a lot of new features. Version 1 is still included and available still via mdv though, in order to not break existing tools.
    For version 1 documentation head over here.

    Installation¤

    # within your conda/virtual env of choice:
pip install mdv
mdv2 -h

    Version 1¤

    Version 1 also contained in the package, unchanged:

    mdv -h
    Throughout this documentation, except the page about version 1, we refer to the new version with "mdv".

    Windows¤

    Additional install:

    pip install colorama # provides ANSI escape color mappings

    Docker¤

    A Dockerfile is in the assets directory of the repo.
